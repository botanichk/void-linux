# Void Linux

![Void Linux Logo](https://upload.wikimedia.org/wikipedia/commons/0/04/Void_Linux_logo.svg)

Minimalistic, independent Linux distribution based on `xbps` package manager.  
Fast, rolling release, systemd‑free.
